# Make It (Not) Gamble
Make it (Not) Gamble is a game made in Godot 4 about cheating as a casino. 

This is our first godot project, made for the "The Very Serious Juniper Dev Game Jam", and can be played [here](https://vulcanohex.itch.io/make-it-not-gamble)
